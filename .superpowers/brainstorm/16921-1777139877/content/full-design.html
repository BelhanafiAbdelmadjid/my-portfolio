01 · Overview

The Usage Agent answers end-user questions about application navigation and features using a RAG pipeline over ingested specification documents. It routes each query through the most relevant knowledge source before synthesising a plain-language response.

02 · How it works
1

User submits a natural language question ("How do I invite a user?")

2

Agent selects the best search tool: specs, domain, frontend, backend, or knowledge

3

Chroma DB returns the top-k embedded document chunks

4

GPT-4.1 synthesises a concise, step-by-step answer from the retrieved context

03 · Key capabilities

▸  Five search tools (specs, domain, backend, frontend, knowledge)

▸  Responds strictly from retrieved context, never hallucinating

▸  Step-by-step navigation instructions for end users

▸  Gracefully admits when context is missing

▸  Integrates as a Nuxt runtime plugin via @brickcode/lib-ai
